# PDF格式完全学习网站

这是一个关于PDF文件格式的纯静态学习网站。

## 项目结构

```
pdf-tutorial/
├── index.html          # 主页面
├── css/
│   └── styles.css     # 样式文件
└── assets/
    └── images/        # 图片资源目录
```

## 功能特性

### 内容覆盖
- **PDF基础** - 介绍PDF的基本概念、历史和优势
- **内部结构** - 详细解析PDF文件结构和对象类型
- **最新特性** - PDF 2.0及后续版本的技术更新
- **安全机制** - 加密、数字签名和权限控制
- **AI集成** - 智能文档分析和自动化工作流
- **工具资源** - 开源库、商业工具和API服务

### 技术特点
- 纯静态HTML/CSS，无需服务器
- 响应式设计，适配所有设备
- 现代化UI设计，使用CSS Grid和Flexbox
- 平滑滚动和动画效果
- 代码示例展示

## 使用方法

### 在浏览器中打开

```bash
# 直接打开index.html文件
open index.html
```

或者双击 `index.html` 文件即可在浏览器中查看。

### 使用本地服务器（推荐）

```bash
# 使用Python启动简单HTTP服务器
python3 -m http.server 8000

# 或使用Node.js的http-server
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 自定义和扩展

### 修改主题颜色

在 `css/styles.css` 中修改CSS变量：

```css
:root {
    --primary-color: #2563eb;
    --accent-color: #f59e0b;
    /* ... */
}
```

### 添加新内容

在 `index.html` 中复制现有的section结构，修改内容即可。

### 添加新页面

1. 创建新的HTML文件（如 `about.html`）
2. 复制 `index.html` 的head和footer部分
3. 添加自定义内容
4. 在导航中添加链接

## 性能优化建议

1. 图片优化：使用WebP格式，压缩大小
2. CSS精简：移除未使用的样式
3. 启用GZIP压缩（服务器配置）
4. 使用CDN加速资源加载
5. 添加缓存头（服务器配置）
